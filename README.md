# BarDock

BarDock is a small Dozer/Hidden Bar-style macOS menu bar hider.

It does one thing:

- click `›` to hide menu bar icons to its left
- click `‹` to show them again
- right-click `›/‹` for settings
- hold Command and drag `·` to choose the hidden boundary

No Screen Recording permission. No Accessibility permission. No icon scanning.

## App

Download the ready-to-use app:

```text
https://github.com/bailegebaiya/BarDock/releases/latest/download/BarDock.zip
```

The built app bundle is also included in this repository:

```text
dist/BarDock.app
```

## Build

```sh
./Scripts/build_app.sh
```

The app bundle is created at:

```text
dist/BarDock.app
```
